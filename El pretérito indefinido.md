>[!note] Used for actions *completed in the past*
### -AR verbs

|                     | **-AR** | **Hablar**   |
| ------------------- | ------- | ------------ |
| yo                  | -é      | habl*é*      |
| tú                  | -áste   | habl*aste*   |
| él/ella/usted       | -ó      | habl*ó*      |
| nosotros/as         | -amos   | habl*amos*   |
| vosotros/as         | -asteis | habl*asteis* |
| ellos/ellas/ustedes | -aron   | habl*aron*   |
